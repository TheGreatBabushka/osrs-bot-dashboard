# osrs_bot_dashboard

A dashboard for an Old-School Runescape bot farm.

## Features

- **Settings Widget**: Configure the API server IP address through a settings dialog accessible from the app bar
- **Persistent Configuration**: API IP settings are saved locally and persist across app restarts
- **Account Management**: View and manage bot accounts
- **Activity Monitoring**: Track bot activity and status

## Configuration

To configure the API server:

1. Click the settings icon (⚙️) in the app bar
2. Enter the API server URL (e.g., `http://192.168.1.100:8080`)
3. Click "Save" to apply the changes

The default API server is `http://localhost:8080`.
